# CourtConnect
A website that connects players and coaches.


Users: volleyball athletes and coaches

User Journey (Athletes): 
Login
Upload highlight video
Include the targeted level of play
Personal information (height, weight, vertical jumps)
Attach sportsRecruits/University Athlete link
Provide contextual information (position, past experience)
Submit to review

Review of feedback (thumbs up/down)
Coaches with more thumbs up get assigned more players to evaluate

User Journey (Coaches):
Register
Provide volleyball/coaching experience, familiarity with each level of play, experience with recruited athletes
Login
List of suggested players
Overview profile and provide assessments

Report:
Recommended level of play (measured on a metric to track distance from the targeted level of play)
Evaluation of different dimensions based on position (spider web chart)
Spider chart with averaged dimension values
Access the individual feedback of each coach and their evaluation
Suggested practice plan/next steps to reach targeted level of play

Second Report:
Re-evaluate new video
Coaches can only access previous feedback (not recommended level of play)
