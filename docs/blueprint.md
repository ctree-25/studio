# **App Name**: CourtConnect

## Core Features:

- Player Profile Creation: Players create profiles with personal information, highlight footage, target level of play (D1, D2, D3), and target schools.
- Coach Assignment: Players are assigned to relevant coaches based on position, level, or region.
- Footage Analysis: AI tool analyzes uploaded player highlight footage, identifying strengths and areas for improvement, using prompts informed by player's target level of play and preferred schools. It outputs the information so that the Coach can incorporate into feedback.
- Coach Feedback: Coaches review player profiles and provide assessments and feedback, tailored to the player's target level of play.
- Anonymized Feedback Delivery: Delivers coach feedback to players anonymously.
- Player View: Dedicated interface for players to manage their profiles, view feedback, and track progress.
- Coach View: Dedicated interface for coaches to review assigned player profiles and provide feedback.

## Style Guidelines:

- Primary color: #6A5ACD (Slate Blue) - a balanced color suitable to athletics but sophisticated, also reminiscent of school colors.
- Background color: #E6E6FA (Lavender) - very desaturated, providing a light and clean backdrop that keeps attention on the content.
- Accent color: #9370DB (Medium Purple) - used for interactive elements, complementing the primary color to guide user actions.
- Body and headline font: 'Inter', a sans-serif font for clear readability on various devices.
- Use clear and modern icons related to volleyball actions and profile elements.
- Clean and organized layout, emphasizing key information and ease of navigation for both players and coaches.
- Subtle transitions and animations to enhance user experience without being distracting.
